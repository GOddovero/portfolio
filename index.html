Proyectos

Chefcito

Proyecto final de la carrera Licenciatura en informatica. El objetivo del proyecto fue incluir inteligencia artificial para mejorar un proceso, el proceso elegido fueron las RECETAS y como se buscan. Tradicionalmente se busca primero la receta para luego ver los ingredientes que son necesarios, en el caso de "Chefcito" es al reves, se agregan los ingredientes que el usuario posee en una base de datos y por medio de una API de Groq se relacionan para generar recetas, sumado a eso se uso Dall-e para generar imagenes.

Tecnologias Utilizadas
Pokedex Reloaded

Con el objetivo de contribuir a la comunidad del juego "Pokemon Reloaded" realizado por F3R se comenzo la creacion de una pokedex donde poder buscar todos los pokemones disponibles en el juego para capturarlos. Sumado a eso se incluyeron opciones de busqueda sujerida por usuarios, ademas se añadieron opciones de intereces para ayudar aún más a los jugadores, como la ubicacion de algunos objetos, formas de evolucionar o pokemones dificiles de conseguir. Ademas en cuanto al aprendizaje que obtuve al momento de desarrollar esta pagina, que aún sigue en desarrollo, fue creada toda en HTML, CSS y JS puro, sin uso de librerias externas, ademas de ser programado completamente en "Project IDX".
